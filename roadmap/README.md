# Roadmap

This directory contains various roadmap and planning documents for the WorldArchitect.AI project.

## Files

1. **amazon_mvp_design_doc.md** - Design document for an Amazon.com MVP replication, created as part of cerebras script testing
2. **cerebras_script_enhancement_summary.md** - Summary of enhancements made to the cerebras script
3. **cerebras_complete_enhancement_summary.md** - Complete summary of all cerebras script enhancement work

## Purpose

The Amazon MVP design document was created as part of testing the cerebras_direct.sh script to demonstrate its capability to generate comprehensive system design documents. The other documents summarize the enhancements made to the cerebras script, particularly the addition of the --light mode flag.
