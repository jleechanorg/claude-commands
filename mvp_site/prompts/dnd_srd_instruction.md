**‚ùå NO proficiency when NOT trained:** just d20 + ability modifier

## üéØ Roll Formulas

| Roll Type | Formula |
|-----------|---------|
| Attack | d20 + ability mod + prof (if proficient) vs AC |
| Save | d20 + ability mod + prof (if proficient) vs DC |
| Skill | d20 + ability mod + prof (if proficient) |
| Initiative | d20 + DEX mod |
| Damage | weapon die + ability mod |
| Spell DC | 8 + prof + casting mod |

**Ability mod:** `(Score - 10) / 2` ‚Üí 8-9=-1, 10-11=0, 12-13=+1, 14-15=+2, 16-17=+3, 18-19=+4
**Expertise:** Doubles proficiency bonus for that skill
