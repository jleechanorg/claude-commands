# World Directory - Campaign World Content

## Overview

This directory contains world content and lore for the default campaign setting used by WorldArchitect.AI. The world content provides rich background material for AI-generated campaigns, including characters, factions, locations, and story elements.

## World Content Files

### world_assiah_compressed.md
- **Purpose**: Main world content for the Assiah setting
- **Key Features**:
  - Detailed world background and history
  - Character dossiers and faction information
  - Geographic locations and descriptions
  - Cultural and political context

### banned_names.md
- **Purpose**: List of restricted names for content filtering
- **Key Features**:
  - Inappropriate name prevention
  - Content moderation support
  - Character name validation
  - Safe content generation

### archive/ Directory
- **Purpose**: Historical and archived world content
- **Contents**:
  - **character_merge_analysis.md** - Character integration analysis
  - **characters_enhanced.md** - Enhanced character descriptions
  - **deletion_analysis.md** - Content removal tracking
  - **factions_enhanced.md** - Detailed faction information
  - **geography_enhanced.md** - Expanded geographic content
  - **timeline_consolidated.md** - World timeline and history
  - **world_mechanics_enhanced.md** - Enhanced world mechanics

## World Integration

### AI System Integration
- World content is loaded into AI system instructions when default world is enabled
- Provides rich context for story generation
- Ensures consistent world building across campaigns
- Supports deep lore references and character interactions

### Content Structure
- **Characters**: Detailed character dossiers with backgrounds and motivations
- **Factions**: Political and social organizations with goals and conflicts
- **Geography**: Locations with descriptions and significance
- **History**: Timeline of events and world development
- **Mechanics**: World-specific rules and systems

## Key Features

- **Rich Lore**: Detailed world background for immersive campaigns
- **Character Depth**: Well-developed NPCs with complex motivations
- **Political Intrigue**: Faction conflicts and power struggles
- **Geographic Diversity**: Varied locations with unique characteristics
- **Historical Context**: Deep timeline providing campaign hooks

## Usage

### Campaign Creation
- Default world content is optionally included in new campaigns
- Provides instant rich setting without requiring user world-building
- AI references world content for consistent storytelling
- Characters and factions can be referenced throughout campaigns

### Content Moderation
- Banned names list prevents inappropriate content
- Content filtering ensures appropriate gameplay
- Safe content generation for all users
- Automated content validation

## World Setting: Assiah

The default world setting focuses on the Celestial Wars and the world of Assiah, featuring:
- **Political Complexity**: Multiple factions with competing interests
- **Character Depth**: Rich NPCs with detailed backgrounds
- **Moral Ambiguity**: Complex ethical situations and choices
- **Epic Scope**: Large-scale conflicts and world-shaping events
- **Player Agency**: Meaningful choices that affect the world

## Maintenance

- World content is version controlled and updated regularly
- Archive directory preserves historical content versions
- Content is tested with AI system for consistency
- Regular review ensures content quality and appropriateness
