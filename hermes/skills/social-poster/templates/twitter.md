# Twitter / X Template

**Character limits:**
- Single tweet: **280 chars** (URLs auto-shorten to 23 chars)
- Long-form (X Premium): 25,000 chars
- Thread: no hard limit but reader fatigue kicks in at 7-10 tweets

**Voice:** Punchy. One idea per tweet. Conversational. Line breaks > punctuation.

**Hashtag rules:**
- **Never** in body of first tweet (kills reach)
- **Only** in last tweet of a thread (and only 1-2 max)
- Maximum 2 hashtags total per thread

## Template — Single tweet

```
{{HOOK_LINE}} ({{CONTEXT_PHRASE}}).

{{ONE_DETAIL}}.

{{LINK}}
```

## Template — Thread (5 tweets)

**Tweet 1/5 (hook):**
```
{{CONTROVERSIAL_OR_SURPRISING_STATEMENT}}.

({{TEASER_OF_THREAD}})

🧵👇
```

**Tweet 2/5 (problem):**
```
The problem:

{{3_BULLETS_WITH_LINE_BREAKS}}
```

**Tweet 3/5 (what we did):**
```
What we built:

{{3_BULLETS_WITH_LINE_BREAKS_AND_NUMBERS}}
```

**Tweet 4/5 (proof):**
```
{{ONE_NUMBER_OR_RESULT}}.

{{ONE_QUOTE_FROM_A_USER_OR_LOG_LINE}}
```

**Tweet 5/5 (CTA):**
```
Repo + docs: {{LINK}}

If this is useful: bookmark, quote-tweet with your stack, or just say hi 👋

#{{TAG_1}}
```

## Anti-patterns

- ❌ Hashtag in first tweet
- ❌ More than 2 hashtags total
- ❌ Threads > 7 tweets (engagement drops)
- ❌ "RT if you agree" / "Follow for more"
- ❌ Putting the link in the first tweet (algorithm deprioritizes)
- ❌ Pure emoji tweets with no substance
