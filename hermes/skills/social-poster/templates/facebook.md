# Facebook Post Template

**Character limits:**
- Post: **63,206 chars** (effectively unlimited; <500 is optimal)
- Optimal length: **80-150 chars** for engagement; **1,000-2,500 chars** for thought-leadership

**Voice:** Friendlier than LinkedIn. Personal. Stories > announcements.

**Media:**
- Posts with images get 2.3x engagement
- Posts with video get even more
- Links auto-preview with OG metadata — make sure your link has og:image, og:title, og:description

## Template — Short (engagement-optimized)

```
{{HOOK_OR_QUESTION}}.

{{ONE_SENTENCE_CONTEXT}}.

{{LINK_WITH_PREVIEW}}
```

## Template — Long-form (announcement)

```
{{STORY_OPENING_WITH_PERSONAL_DETAIL}}.

{{THE_TURNING_POINT_OR_PROBLEM}}

{{WHAT_YOU_BUILT_OR_LEARNED}}

{{ONE_NUMBER_OR_QUOTE}}

{{CTA_OR_LINK}}

#{{TAG_1}} (optional, only 1)
```

## Anti-patterns

- ❌ Pure share with no commentary
- ❌ Engagement bait ("Type YES if you agree")
- ❌ Cross-posting the LinkedIn post verbatim without adapting tone
- ❌ Links without OG metadata (looks broken)
