# Threads (Meta) Template

**Character limits:**
- Post: **500 chars max** (hard limit)
- Comments on threads: 500 chars

**Voice:** Casual, conversational. First-person. Closer to Twitter than LinkedIn. Threads (the platform) rewards authenticity over polish.

**Hashtags:** 1-3 max, can be inline.

## Template

```
{{CASUAL_HOOK_LINE}}.

{{CONTEXT_OR_BACKSTORY_IN_2-3_SENTENCES}}.

{{DETAIL_OR_TURNING_POINT}}.

{{LINK_IF_ANY}}
```

## Example (filled)

```
shipping open source is the best forcing function for writing docs.

we released {{PROJECT}} last week — an AI agent harness with a deploy pipeline,
skill framework, and 7-green verification baked in. zero users, lots of dogfood.

the part nobody tells you: the first 10 PRs were all docs.

{{LINK}}
```

## Anti-patterns

- ❌ Corporate / LinkedIn-voice posts ("Thrilled to announce…")
- ❌ >500 chars (hard-truncated, looks broken)
- ❌ Hashtag-heavy (this isn't Instagram)
- ❌ Cross-posting verbatim from LinkedIn / Twitter
