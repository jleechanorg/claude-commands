# /benchg [project_number] [custom_description]

Comprehensive benchmark comparison between Genesis and Ralph orchestration
systems — runs both in isolated tmux sessions, monitors, collects metrics,
runs a consensus code review, live build/test validation, and produces a
summary report.

Read `~/.claude/skills/benchg/SKILL.md` and execute the full workflow with
the provided project number or custom description.

## Usage

```
/benchg 1                                     # CLI Text Processing Utility
/benchg 2                                     # REST API Web Service
/benchg 3                                     # Data Processing Pipeline
/benchg custom "your custom project"          # Custom project spec
```
